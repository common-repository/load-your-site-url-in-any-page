Duplicate Front Page,Post easily by thier url with our shortcode and page template.

== Description ==

Duplicate Front Page,Post easily by thier url, duplicate front page will create one page template called **Duplicate Front Papge** when will you set that template you will find below one meta field there have to enter **shortcode** as *[duplicate-front-page-by-url url="url of page/post you want to copy"]*


== Installation ==

Upload the 'Duplicate Front Page' plugin to your blog, Activate it, then visite plugin setting > how to display.

e.g.

1. Upload `load-page-or-post-by-url.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why plugin not working or loading URL? =

 - Plugin will not work on local server. You should have an online website for the plugin to function properly

= Why error or permission issue? =
 
 - When you loading any page or url, That page should be public Or everything loading without restriction( like css,js,images ).

= When result loading only or timeout? =
 
 - Loading same URL in same page will break functionality.
